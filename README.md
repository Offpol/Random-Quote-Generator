# Random-Quote-Generator
## Автор: [ЕГОР СКЛЯРОВ]
## Описание:
Графическое приложение для генерации случайных цитат. Позволяет просматривать предустановленные цитаты или добавлять свои. История сгенерированных фраз сохраняется в файл `quotes.json` и может быть отфильтрована по автору или теме.

## Технологии:
*   **Язык:** Python 3.x
*   **Библиотеки:** `tkinter` (стандартная), `random`, `json`
*   **Система контроля версий:** Git

## Требования к запуску:
1. Установить Python (версия >= 3.6).
2. Библиотека `tkinter` входит в стандартную поставку Python.
3. Установить Git (для работы с репозиторием).

## Инструкция по запуску:
1. Клонируйте репозиторий:
   `git clone <ссылка_на_репозиторий>`
2. Перейдите в папку проекта.
3. Запустите приложение:
   `python main.py`

## Возможности приложения:
1.  **Генерация:** Кнопка «Сгенерировать цитату» выбирает случайную фразу из базы.
2.  **История:** Все сгенерированные цитаты отображаются в списке справа.
3.  **Фильтр:** Можно отфильтровать историю по «Автору» или «Теме».
4.  **Добавление:** Пользователь может добавить свою цитату (проверка на пустые поля).
5.  **Сохранение:** История и база цитат сохраняются в файл `quotes.json` при каждом действии.
